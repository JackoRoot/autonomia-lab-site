Calculs et sources
Conso totale jour = WC + Hygiène + Vaisselle + Linge + Cuisine + Entretien + Jardin Eau potable = Hygiène + Cuisine + Entretien (≈ 57% en moyenne) Eau technique = WC + Vaisselle + Linge + Jardin (peut être eau pluie) Cuve recommandée = Conso × jours d'autonomie
  • Conso WC : 9-12 L (anciens), 6-9 L (standard), 3/6 L (double débit), 0 L (sèches) - sources ADEME, Geberit, Planetoscope
  • Conso douche : 15-20 L/min (standard), 6-8 L/min (économe), Selectra/ADEME
  • Bain : 150-200 L (ADEME)
  • Lave-linge : 40-50 L (récent), 70-90 L (standard), 100-120 L (ancien) - BUT, ADEME, Selectra
  • Lave-vaisselle : 9-12 L (récent), 16-20 L (ancien), 30-60 L (à la main robinet ouvert) - ADEME, Selectra, Hydrao
  • Jardin : ~10 L/m² selon ADEME (varie selon saison et plantation)
  • Cuisine + boisson : 7% conso totale soit ~10 L/personne/jour - CIEAU
  • Conso enfant : 55 L/jour vs 148 adulte - moyenne SISPEA 2024
  • Répartition potable/technique : calculée à partir des % CIEAU (hygiène + alim + entretien = ~57% / WC + lessive + vaisselle + jardin = ~43%)
